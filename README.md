## Vrednosti
* Prometni odsek
* MO = Motorji
* OV = Osebna vozila
* AB = Avtobusi
* LT = Lahka tovorna vozila (< 3,5t)
* ST = Srednja tovorna vozila (3,5t - 7t)
* TT = Tezka tovorna vozila (nad 7t)
* TP = Tovor s prikolico
* VL = Vlacilci
* Vsa vozila
